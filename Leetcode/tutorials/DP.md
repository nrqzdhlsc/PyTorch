动态规划主要是使用 [递归](https://www.geeksforgeeks.org/recursion/)的优化问题。无论何时我们看到递归问题在同样的输入上有重复调用，我们就可以使用动态规划。想法就是存储子问题的结果，因此我们不用重复计算。这种简单优化就能将时间复杂度从指数级减少到多项式级别。比如，如果我们写一个简单的斐波那契数列解法，时间复杂度是指数级别。如果通过存储子问题来优化，整体复杂度变为线性。

```c++
// 递归解法：指数级复杂度
int fib(int n) {
    if (n <= 1) {
        return n;
    }
    return fib(n - 1) + fib(n - 2);
}

// DP解法：线性复杂度
f[0] = 0;
f[1] = 1;
for (int i = 2; i <= n; i++) {
   f[i] = f[i-1] + f[i-2]; 
}
return f[n];
```
